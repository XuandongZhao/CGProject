# CGProject

Project on Computer Graphics. Simulate aircraft flying in the city and projecting missiles for bombing. 

## Highlight

- Multi-Light Rendering
- Collision detection
  - OBB collision detection
  - Half-plane intersection
- Ocean Waves
- Particle System
  - Explosion
  - Fire
